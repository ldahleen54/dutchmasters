# [Dutchmasters](https://dutchmasters.org)
Website for a gun tournament

# Setup
1. clone the repo
2. `cd gatsbynetlify`
3. `npm install -g gatsby-cli` or use another way to install the gatsby cli
4. `npm install`
5. `gatsby develop`
6. The default port is 8000

# Planned Features
* Sort pictures by year
* Search scores by name and sort by highest scores
* Average for each score type by person
* Content management system
